# Library Management System

This project implements a library management system using C++ and object-oriented programming principles. The system allows for adding books, borrowing and returning books, tracking transactions, and searching for books by category.

## Features

- Book Management
- User Management
- Transaction Tracking
- Data Persistence with file handling

## Building

To build the project, use the following steps:

1. Install [CMake](https://cmake.org/install/).
2. Run `cmake .` in the root directory.
3. Build the project with `make`.

## Usage

Run the project with `./LibraryManagementSystem`.
